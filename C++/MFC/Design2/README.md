# Design2 设计2
## 设计任务
设计基于文档/视图结构的MFC应用程序，在窗口中绘制一个椭圆形轨道，并在轨道上绘制一个小圆，轨道用红色粗实线画笔绘制，小圆用黄色画笔绘制并将内部填充为黄色，小圆的圆心在轨道上并沿轨道匀速运动。

## 设计要求
1. 设计“启动”、“停止”菜单项控制小圆的运动状态。
2. 设计菜单项“逆时针”、“顺时针”，用来控制小圆沿轨道运动的方向。

## 设计思路
1. 绘制椭圆需要用到Ellipse函数，要知道其左上角与右下角的坐标，即两个坐标。对于在椭圆上匀速运动的圆来说，圆心的运动轨迹同椭圆的轨迹相重合，即圆心运动方程遵循椭圆的参数方程。由于椭圆的中心不在原点，因此还需要椭圆的中心坐标才可确定圆心的运动轨迹，需要定义数组长度为3的CPoint类型数组point。

2. 通过几何关系确定椭圆的相关参数，已知左上角和右上角坐标，可用中点坐标公式求得椭圆的中心，进而得出椭圆的参数方程。定义其参数为变量alpha，若alpha均匀变化，则圆心沿着椭圆轨迹匀速运动，需要设置定时器来体现变化。

3. 使用红色画笔绘制椭圆，保持椭圆轨迹不变。由于绘制圆形需要圆的半径，因此定义半径r，用黄色画笔绘制圆的边界，并用黄色画刷填充圆。

4. 在资源视图上添加启动、停止、逆时针、顺时针的菜单项，并赋予相应ID，其中启动与停止、逆时针与顺时针分别是两对相反的菜单项。对相关菜单项进行操作时，指定：若启动项选中，则停止项不可选；若逆时针项选中，则顺时针项不可选。对于工具栏按钮的使用，只需绑定相应状态的ID即可。

5. 在相应菜单项对应的函数中编写代码控制圆的运动状态。

## 变量定义说明
1. 控件ID

ID|Caption
-|-
ID_START|开始运动
ID_STOP|停止运动
ID_NEG_ROT|逆时针
ID_POS_ROT|顺时针

2. 变量
int m_nStart 启动:1 停止:0 用于控制圆的开始运动与停止运动

int m_nIsNeg 逆时针:1 顺时针:0 用于控制圆运动时的顺逆时针方向

CPoint point[3] [0]:椭圆中心点坐标 [1]:椭圆左上角坐标 [2]:椭圆右下角坐标

double alpha 椭圆参数方程的角参数，其均匀变化表征圆在椭圆上匀速运动

int r 圆的半径，绘制圆形必备的参数
