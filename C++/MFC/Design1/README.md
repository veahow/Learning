# Design1 设计1
## 设计任务
设计基于对话框的MFC应用程序，在窗口中间位置绘制一个正五边形，正五边形使用红色画笔绘制，并且将正五边形的内部用红色进行填充。设计两个方向按钮（向左、向右），用户按下的两个方向按钮时，正五边形沿着方向按钮对应的方向移动。
## 设计思路
1. 绘制多边形需要用到Ploygon函数，用该函数绘制多边形需要两个参数：CPoint类型的坐标数组和数组长度。由于要绘制正五边形，因此只需要定义长度为5的CPoint类型坐标数组，即
`CPoint m_Ord[5];`
m_Ord表示正五边形的五个顶点，并含有数据成员x与y，分别对应点的横坐标和纵坐标，因此绘制正五边形的代码为
`Ploygon(m_Ord, 5);`
因为未确定m_Ord数组中各点的坐标值，所以需要计算出各点的值，才可进行正五边形的绘制。
2. 若要根据正五边形的性质得其五个顶点的坐标值，还需要正五边形的长度，定义为m_dLen。初始化正五边形的某一个顶点（本设计中初始化m_Ord[0]）的值，由正五边形的性质可以计算出其他四个顶点的坐标，此时便可以根据m_Ord绘制出正五边形。
3. 绘制正五边形之前，需要定义画笔和画刷。画笔用于绘制正五边形的边，画刷用于填充正五边形的内部，两者颜色均指定为红色，即RGB(255, 0, 0)。此时，便可绘制出完全红色的正五边形。
4. 根据程序运行情况进行调试，调整正五边形顶点m_Ord[0]的初始化数值，直至正五边形位于窗口中央。由于其他顶点的坐标值是由m_Ord[0]计算出来的，因此更改其初始化数值后，其余四个顶点的值也会改变，使正五边形的形状保持不变（前提是m_dLen不变，本设计中该变量始终保持不变）。
5. 在MFC程序的资源视图中设置两个按钮，并将其绑定为一组按钮：
ID|Caption|GROUP
-|-|-
IDC_RIGHT|→|TRUE
IDC_LEFT|←|FALSE
对按钮绑定函数，每次点击按钮，产生对正五边形的位移操作，每次位移移动m_nUnitStep个单位。该变量保持不变，表示每次位移移动固定单位。进行位移操作时，只需对顶点横坐标进行加减操作，加为右移，减为左移。
6. 定义边界数组m_Border限定刷新区域，只对正五边形运动区域进行刷新操作，以提升程序效率。
